# CLI Calculator

A simple command-line calculator that performs basic arithmetic operations.

## What it does

The program asks the user for two numbers and an operation (addition, subtraction, multiplication, or division), then prints the result. It also handles division by zero and invalid operations without crashing.

## How to run

```bash
python cli_calculator.py
```

## Example usage

```
=== Calculator ===
Available operations: +, -, *, /
Enter the first number: 10
Enter the operation (+, -, *, /): /
Enter the second number: 2
Result: 5.0
```
Division by zero example:

```
Enter the second number: 0
Error: cannot divide by zero.
```
